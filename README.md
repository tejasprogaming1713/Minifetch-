Lightweight system information tool written in **Bash** inspired by Neofetch.

---

## Features
- OS information
- Kernel version
- CPU info
- RAM usage
- Disk usage
- Public IP
- Lightweight and fast

---

## Install

```bash
curl -s https://raw.githubusercontent.com/tejasprogaming1713/Minifetch-/main/minifetch -o /usr/local/bin/minifetch
chmod +x /usr/local/bin/minifetch
```

---

## Usage

```bash
minifetch
```
